# Developer's Portfolio

A personal portfolio website built with React, hosted on GitHub Pages.

**Live Site:** https://apoorvverma.github.io/Portfolio/

## Tech Stack

- React.js
- Bootstrap
- React Typist
- GitHub API

## Quick Start

1. Clone the repo
2. Edit `src/editable-stuff/configurations.json`
3. Add your resume as `src/editable-stuff/resume.pdf`
4. Run `npm install && npm start`

## Deployment

```bash
npm run deploy
```
